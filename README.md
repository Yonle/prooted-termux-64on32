# prooted-termux-64on32
A forked prooted termux from https://github.com/Neo-Oli/chrooted-termux for 32 bit Android OS user on 64 Bit device.

## Running
```bash
# Prepare everything & Install rootfs
./prooted-termux

# Launch your 64 bit Termux
./start.sh
```
